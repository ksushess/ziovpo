Тема проекта:

Сервис онлайн-заказа фото-печати - для управления заказами на печать фотографий различных форматов с доставкой.

Основые сущности:

Customer (клиент) - информация о покупателе(имя, фамилия, телефон, почта); связь с заказами клиента.

Format (формат фото) - параметры фото(10х15, А4 и т.д.); название, описание и цена.

Photo (фото) - информация о фото и свзяь с форматом и заказом.

Order (заказ) - основная сущность заказа; статус заказа, общая стоимость, дата создания, связь с клиентом, фото и доставкой.

Delivery (доставка) - информация о доставке, адрес, трек-номер, статус.

Операции сервиса:

CRUD  операции для создания, чтения, удаления и обновления, управление всеми сущностями, то есть заказами, доставкой и т.д.

Бизнес операции: создание заказа с автоматическим расчетом стоимости на основе выбранных форматов; обновление статуса заказа и доставки; получение истории заказов клиента; статистика по популярным форматам; общая статистика по заказам и доходу.
